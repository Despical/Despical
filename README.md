## Documentation

My documentation website for project guides, setup notes, configuration references, and usage examples. It is the main place where I keep longer-form documentation that does not fit cleanly inside a repository README.

- https://docs.despical.dev

## Libraries

A static dependency hub for my reusable libraries, with copyable Maven and Gradle snippets. It is meant to make integration details easy to find without digging through release pages or individual repositories.

- https://libs.despical.dev

## Javadocs

Centralized generated API documentation for my open-source Java projects. This keeps package references, event APIs, and public method documentation available in one place across multiple repositories.

- https://javadoc.despical.dev

## TikFetch

TikFetch is an ultra-clean, lightning-fast web application designed to save public TikTok videos, photo posts, swipeable photo galleries, and MP3 audio without watermark clutter, annoying popups, or ad-heavy distractions.
- https://tikfetch.despical.dev
